## Format

Each example file follows this structure:
```markdown
# [Video Title]

**Source:** [YouTube Link](https://youtube.com/watch?v=...)
**Style:** [Summary Style]
**Provider:** [AI Provider]
**Generated:** YYYY-MM-DD

---

[Full summary output]
```

## Adding Your Own Examples

1. Watch a YouTube video with captions
2. Use CapSummarize to generate a summary in your desired style
3. Copy the output
4. Create a new `.md` file following the naming convention: `{style-name}.md`
5. Add the generated summary with metadata (Source, Style, Provider, Date)

## Tips for Good Examples

- Choose videos with clear captions (YouTube recommended)
- Pick content that showcases each style's strengths:
  - **Technical** works best for coding/tech tutorials
  - **Educational** works best for learning-focused content
  - **Casual** works best for entertaining/personal content
  - **Executive** works best for business/industry news
  - **Marketing** works best for product launches/announcements
- Include the YouTube link so others can verify and learn from the same source

## Style Descriptions

- **Default** - Balanced overview for general audiences
- **Educational** - Structured learning with objectives and scholarly analysis
- **Technical** - Detailed analysis with code and implementation details
- **Casual** - Friendly, conversational summary with personality
- **Executive** - Concise decision-focused briefing for leaders
- **Marketing** - Benefit-driven summary with call-to-action
- **News** - Objective, journalistic reporting with facts
- **Podcast** - Episode recap with quotes and timestamps
- **Kids** - Simple explanation with activities for ages 7-12
- **Blog** - SEO-optimized post with sections and engagement
- **YouTube** - Timestamped chapters with navigation links
- **Cheatsheet** - Technical reference with commands and examples
- **Recap** - 60-second interview-style summary
- **Interview** - Interview-prep summary with Q&A and examples
- **X/Twitter** - Viral Twitter thread with hooks and engagement
