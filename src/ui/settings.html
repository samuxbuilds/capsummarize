View import file format

Import files should be JSON with this structure: 

{
  "version": 1,
  "variants": [
    {
      "variant": "my-style-id",
      "description": "A brief description",
      "prompt": "Your prompt with {transcript}",
      "outputType": "text",
      "imageStyle": "optional style",
      "isThumbnail": false
    }
  ]
}

variant: Unique ID (lowercase, numbers, hyphens)

outputType: "text", "image", or "video"

imageStyle: Style description (for image output)

isThumbnail: true/false (for image output)
